This is the PowerShell script to set up a 'Maintenance Window'. Make sure to customize all variables according to your needs, such as:
apiToken, environmentUrl, name, description, maintenanceType, suppression, disableSyntheticMonitorExecution, and so on. Also, some useful documentation links are below.
https://www.dynatrace.com/support/help/observe-and-explore/notifications-and-alerting/maintenance-windows/define-maintenance-window
https://www.dynatrace.com/support/help/dynatrace-api/environment-api/maintenance-windows
